# Security Policy

## Scope

This repository contains AI agent definitions, prompts, and workflow templates.
It does not include executable application code, network services, or data stores.

## Reporting a Vulnerability

If you discover a security issue — including sensitive information accidentally
committed, prompt injection risks, or unsafe default configurations — please
report it by opening a GitHub issue marked **[SECURITY]** or by contacting the
maintainer directly via GitHub.

Do not include sensitive details in public issues. Use GitHub's private
vulnerability reporting if available.

## Expectations

- Reports will be acknowledged within 5 business days.
- Confirmed issues will be addressed before the next release.
- Credit will be given to reporters unless anonymity is requested.

## Safe Content Policy

All content in this repository is designed to be NDA-safe and free of secrets.
See [docs/public-safety.md](docs/public-safety.md) and
[validation/nda-safety-checklist.md](validation/nda-safety-checklist.md).

# Quality & Safety Validation Rules

Checklists and rule files enforcing output quality, NDA safety, and public-release
readiness. Used by review agents and validation workflows.

- `ai-output-validation.md` — general quality gates for AI-generated output
- `code-review-rules.md` — standards applied during code review prompts
- `compactness-rules.md` — token efficiency and conciseness enforcement
- `markdown-quality-rules.md` — documentation formatting standards
- `nda-safety-checklist.md` — PII and company-specific content detection
- `public-release-checklist.md` — pre-publish verification steps

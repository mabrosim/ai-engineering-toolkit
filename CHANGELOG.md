# Changelog

All notable changes to this project are documented here.

---

## [0.1.0] — 2026-07-02

Initial public release.

### Added

- AET Orchestrator, Steward, and specialist agent bundle (Engineering, Documentation,
  Prompt, Review, Validation, Lightweight, Proposal)
- Core prompts: assess, discover, extract, code-review, validate, generate, propose,
  quality-gate, commit-message, review-proposal
- NDA-safety and public-release checklists (`validation/`)
- `aet-install.sh` installer (repo and global modes, content-path bundling)
- Engine instruction files: AI collaboration rules, communication style, documentation style
- Three-tier content model: always-on `instructions/`, on-demand `skills/`, named `catalogs/`
- Full documentation: philosophy, repository rules, decisions log, public safety guide,
  external project usage guide
